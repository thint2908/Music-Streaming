Để chạy chương trình web nghe nhạc trực tuyến cần thực hiện các bước sau:
B1: Lấy folder source trong bài nộp cho vào C:\xampp\htdocs\
B2: Mở xampp, bật apache và mysql
B3: vào http://localhost:8080/phpmyadmin/, tạo database online_music, sau đó nhập
file online_music.sql
B4: vào đường dẫn http://localhost:8080/CuoiKiWeb/source/index.php

sẽ xuất hiện trang index.php, để nghe nhạc hay thực hiện các chức năng khác phải
cần phải đăng nhập.

tài khoản đang nhập có admin và người dùng có trong table account với priority
= 1 là admin và priority = 2 là user
